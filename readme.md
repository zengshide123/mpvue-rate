## mpvue-rate

> 项目用于mpvue开发小程序，是一个rate组件

### Usage

```
    npm i mpvue-rate
```
### 项目说明

- 本项目主要想法来自于[vue-tiny-rate](https://github.com/shengxinjing/vue-tiny-rate)
- 项目文档请到上述链接处查看，基本保持一致
- 增加title（props）用于传递组件左侧文字标题
